# Git Made Easy - Frontend 

This Next.JS frontend application is an interactive platform designed to teach Git through lessons and interactive elements, i.e. tasks, hints, and feedback. It consists of a responsive, intuitive UI paired with a Spring Boot backend system to provide an interactive learning experience.

---

## Overview

Built with Next.js, React and TypeScript, this frontend leverages **clean architecture principles** to ensure reusability and scalability of components and UI patterns.

- **Separation of Concerns:** UI Components, Hooks, Services and external states are individually encapsulated and managed.
- **DRY (Don't Repeat Yourself):** Reusing shared UI components, utility functions and factories for data and objects.

This results in a scalable and testable frontend application that can easily evolve alongside a pairing backend system.

## Getting Started

### Prerequisites

- Node.js 18+
- npm
- Configure environment variables or '.env.local' files

### To install, use `npm install` to install all the required dependencies, then `npm run dev` to run the frontend application.
